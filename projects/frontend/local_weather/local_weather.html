Local Weather Viewer

This application provides users with information about the weather in their locality. The application makes use of the browser's geolocation functionality to provide this information - as such, mobile users can expect to receive a significantly more accurate weather report.

The reported temperature can be converted between Celsius and Fahrenheit at the touch of a button. The application's presentation changes depending on the weather data provided to the user. The application is responsively-designed and consumes JSON data from the OpenWeather API.

Note: Due to limitations inherent to the OpenWeather API, this application must route API requests through a proxy. There may be a delay during certain application actions as a result.
